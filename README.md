# Automação de Planilhas com Python

Projeto para automação de análise de planilhas Excel usando Pandas.

## Funcionalidades
- Junção de planilhas por MSISDN
- Identificação de linhas livres e ocupadas
- Cálculo de gastos por setor
- Geração de planilhas finais com abas

## Tecnologias
- Python
- Pandas
- Excel
